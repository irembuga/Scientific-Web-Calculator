# Çok Temalı Bilimsel Hesap Makinesi

Bu proje, HTML, CSS ve JavaScript kullanılarak geliştirilmiş, web tabanlı çok fonksiyonlu bir bilimsel hesap makinesidir. Kullanıcı deneyimini artırmak amacıyla tamamen özelleştirilebilir renk temalarına sahiptir.

## Özellikler
* **Trigonometrik Hesaplamalar:** Sinüs, kosinüs ve tanjant işlemleri (derece bazlı hatasız hesaplama).
* **Bilimsel Fonksiyonlar:** Logaritma, doğal logaritma (ln), karekök, üs alma ve Pi/Euler sabitleri.
* **Dinamik Tema Motoru:** Pembe, Mavi, Mor, Turuncu, Sarı, Kırmızı, Siyah, Beyaz ve Gri olmak üzere 9 farklı renk teması arasında anlık geçiş.
* **Akıllı Ekran Yönetimi:** Uzun sonuçlarda otomatik yazı boyutu küçültme ve hatalı girişlerde "Hata" koruması.
* **Klavye Desteği:** Numpad ve klavye kısayolları ile tam entegrasyon (Enter, Backspace, Escape).
